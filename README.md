# Редактор на български текст

Локално уеб приложение за корекция на български текст с AI.

Приложението може да:

- поправя правописни грешки;
- поправя граматика;
- добавя и маха препинателни знаци;
- пази оригиналния смисъл на текста;
- показва списък с направените поправки.

## Режими

### Коригирай

Връща само готовия коригиран текст.

### Покажи поправките

Връща коригирания текст и списък с промени:

```json
{
  "original": "...",
  "corrected": "...",
  "changes": [
    {
      "from": "...",
      "to": "...",
      "reason": "Правопис"
    }
  ]
}
```

## Бърз старт на Windows

1. Инсталирай Node.js от:

```text
https://nodejs.org/
```

2. Отвори папката на проекта.
3. Копирай `.env.example` и преименувай копието на `.env`.
4. В `.env` сложи твоя Gemini API ключ:

```text
GEMINI_API_KEY=твоя_ключ_от_Google_AI_Studio
GEMINI_MODEL=gemini-3.5-flash
PORT=8787
```

5. Стартирай:

```text
Стартирай редактора.bat
```

6. Отвори:

```text
http://localhost:8787
```

## Стартиране през PowerShell

```powershell
cd C:\Users\User\Documents\IPTV\bg-text-corrector
node server.js
```

## Без API ключ

Без `GEMINI_API_KEY` приложението пак се отваря, но работи в демо режим. Демо режимът е само за проверка на интерфейса.

## Важно за GitHub

Не качвай `.env` файла в GitHub. В него стои тайният Gemini API ключ.

Файлът `.gitignore` вече е настроен да не качва `.env`.

GitHub Pages не е подходящ за това приложение, ако ще използва Gemini API ключ. Причината е, че GitHub Pages няма защитен сървър за тайни ключове.

За онлайн версия използвай услуга като Render, Railway или VPS. Виж `DEPLOY_RENDER.md`.
