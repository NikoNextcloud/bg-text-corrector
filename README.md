# Редактор на български текст

Локално уеб приложение за корекция на български текст с AI.

## Стартиране

Преди стартиране трябва да имаш инсталиран Node.js. Ако го няма, изтегли го от:

```text
https://nodejs.org/
```

1. Отвори папката `bg-text-corrector`.
2. Задай OpenAI API ключ в същия прозорец:

```powershell
$env:OPENAI_API_KEY="твоя_API_ключ"
```

3. Стартирай приложението:

```powershell
npm start
```

4. Отвори:

```text
http://localhost:8787
```

По желание можеш да смениш модела:

```powershell
$env:OPENAI_MODEL="gpt-5.2"
```

Без API ключ приложението работи в демо режим, за да можеш да разгледаш интерфейса.

Можеш да стартираш и с файла `Стартирай редактора.bat`.
