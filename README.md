# Редактор на български текст

Уеб приложение за корекция на български текст с AI чрез Groq.

Приложението може да:

- поправя правописни грешки;
- поправя граматика;
- добавя и маха препинателни знаци;
- пази оригиналния смисъл на текста;
- показва списък с направените поправки.

## Режими

### Коригирай

Връща готовия коригиран текст.

### Покажи поправките

Връща коригирания текст и списък с промени.

## Бърз старт на Windows

1. Инсталирай Node.js от:

```text
https://nodejs.org/
```

2. Създай Groq API ключ:

```text
https://console.groq.com/keys
```

3. Копирай `.env.example` и преименувай копието на `.env`.
4. В `.env` сложи:

```text
GROQ_API_KEY=твоя_groq_ключ
GROQ_MODEL=llama-3.3-70b-versatile
PORT=8787
```

5. Стартирай:

```text
Стартирай редактора.bat
```

6. Отвори:

```text
http://localhost:8787
```

## Без API ключ

Без `GROQ_API_KEY` приложението пак се отваря, но работи в демо режим. Демо режимът е само за проверка на интерфейса.

## Важно за GitHub

Не качвай `.env` файла в GitHub. В него стои тайният API ключ.

Файлът `.gitignore` вече е настроен да не качва `.env`.

GitHub Pages не е подходящ за това приложение, ако ще използва AI ключ. Причината е, че GitHub Pages няма защитен сървър за тайни ключове.

За онлайн версия използвай Render. Виж `DEPLOY_RENDER.md`.
